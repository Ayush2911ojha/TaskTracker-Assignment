##  Task Tracker Application

A simple Task Tracker web application built using **React**.  
This app allows users to add tasks, view them in a list, and mark tasks as completed using a checkbox.

---

##  Features

- Add new tasks  
- View all added tasks  
- Mark tasks as **Completed** or **Pending** using a checkbox  
- Tasks are saved using browser **localStorage**, so data persists after page refresh  

---

##  Tech Stack

- React (Functional Components & Hooks)  
- CSS  

---

## Installation

1. Clone the repo:
    ```
    git clone https://github.com/Ayush2911ojha/TaskTracker-Assignment
    cd TaskTracker
    ```
2. Install dependencies:
    ```
    npm install
    ```

## Run locally

- Start the development server:
  ```
  npm start
  ```
- Open your browser at http://localhost:5173




## Notes

- Data is stored in the browser so task state persists between sessions.
- Adjust storage key or UI behavior in src as needed.
